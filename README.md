# stock-and-shock
Приложение обвёрнутое в хрустящий фантик Docker'а манит своим изяществом и совращает умы. Состоит из двух страниц:
  * Первая: базовые котировки акций Apple, можно играться с датами начала и конца
  * Вторая: базовый обзор данных о чаевых в определённом формате, нужно будет задать даты для гененарии данных, можно скачать графики, но предупрежданию - в случае если они прожгут вашу сечатку, с меня шиты-крыты, формат принимаемых данных писать тут не хочу - мне лень 

## Инструкции по применению
  * Если вы родились позже 1980-х годов, то всё тут понятно
  * Если вы родились раньше 1980-х годов... я соболезную, вы наверно скучаете по играм с динозаврами

## Инструкции по запуску
1. Запустите терминал в папке репозитория `stock-and-shock`
3. Создайте образ через docker: 
```
   docker build -t stock-and-shock .
```
   Данная команда создаст образ на основе Dockerfile из текущей папки и назовёт его `stock-and-shock`

4. Запустите контейнер на основе данного образа:
 ``` 
    docker run -d -p 8501:8501 stock-and-shock   
```
